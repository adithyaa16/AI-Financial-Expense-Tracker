# 💰 AI Financial Expense Tracker & Analytics

## 📌 Project Overview

AI Financial Expense Tracker is a smart financial analytics application developed using Python and Streamlit.

This application helps users:
- Track expenses
- Analyze spending habits
- Predict future expenses
- Upload financial datasets
- Generate reports
- Send reports through email
- Store previous reports history

---

# 🎯 Features

## 🔐 Login System
Secure login page for users.

## 📊 Dashboard
Shows:
- Average Income
- Average Expense
- Average Savings

## 📈 Expense Analytics
Interactive charts:
- Histogram
- Pie Chart
- Box Plot
- Line Graph

## 💡 Financial Guide
Provides:
- Savings alerts
- Overspending alerts
- Smart suggestions

## 🤖 AI Finance Assistant
Users can ask:
- Highest expense month
- Lowest expense month
- Budget suggestions
- Savings tips

## 📂 Upload Dataset
Users can upload CSV datasets and generate:
- Charts
- Reports
- Analytics

## 📧 Email Reports
Send financial reports directly to email.

## 📁 History
Stores previous uploaded reports.
